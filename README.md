## 环境搭建

- 根据[Flutter中文网](https://flutterchina.club/get-started/install)搭建开发环境，使用Android Studio安装Flutter插件，点击pubspec.yaml的package get，然后运行

## App目录结构
>- |--lib
>    - |-- main (入口类)
>    - |-- loading (启动页)
>    - |-- splash_screen (引导页)
>    - |-- util (工具类)
>    - |-- base (基类，封装基类BaseWidget和BaseWidgetState)
>    - |-- http (网络请求相关类)
>    - |-- common (常用类)
>    - |-- event (事件类)
>    - |-- model (实体类)
>    - |-- ui (界面相关)
>    - |-- util (工具类)
